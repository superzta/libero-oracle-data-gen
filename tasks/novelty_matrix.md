# Novelty Matrix

| Task | Pick/place | Basket/tray/drawer | Open/close | Stove on/off | Stacking | Simple pushing | Novel element |
| --- | --- | --- | --- | --- | --- | --- | --- |
| Peg insertion | Uses grasping | No | No | No | No | No | Precision constrained insertion with pose alignment and tight clearance. |
| Tool sweep | Grasping tool, not target | Dustpan target is container-like | No | No | No | Yes | Tool-mediated non-prehensile manipulation of a separate target object. |
| Button then box | Yes | Box placement is container-like | No | Button analog to appliance interaction | No | No | Ordered compound task with a state change prerequisite before placement. |
| Ring hook | Uses grasping | No | No | No | No | No | Hanging/hooking with post-release support and stability. |

Existing LIBERO categories include pick/place, putting objects in baskets/trays/drawers, opening and closing drawers or microwaves, stove state changes, bowl stacking, and a simple push-the-plate task. The approved tasks add precision insertion, tool use, sequential prerequisite state changes, and hanging.
